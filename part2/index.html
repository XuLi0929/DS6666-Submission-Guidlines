Step 2 of 3 — Organize Submission Directory

All assignment files must be placed inside a single submission folder. The autograder expects a specific directory structure — submissions that don't follow this format will receive a 0.

Create a submission folder and move your main.py into it:

# Create submission directory
mkdir -p ~/ds6666_hw3_submission

# Move your main.py into the submission folder
mv main.py ~/ds6666_hw3_submission/

Your submission directory should look like this:

ds6666_hw3_submission/
├── main.py
├── README.md   (optional)
└── requirements.txt   (if using additional packages)
Grading penalty: Submissions with incorrect directory structure or missing main.py will be automatically rejected by the autograder. Make sure main.py is directly inside the folder, not in a subfolder.

If you have additional files (helper modules, config files), include them in the same directory. Do not include your virtual environment or __pycache__ folders.
